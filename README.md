# TypoSwitch ⌨️

**Switch mistyped text in one click**

TypoSwitch is a Chrome Extension that fixes text typed in the wrong keyboard layout between Hebrew and English.

---

## 🚀 Features

- 🔄 Fix selected text instantly
- ✍️ Fix the last typed word
- ⌨️ Keyboard shortcut support (Ctrl+Shift+Y)
- 🖱️ Right-click context menu
- ⚙️ Settings page
- 💡 Smart detection (basic)
- ✨ Visual feedback (toast)

---

## 🧠 How it works

Sometimes you type with the wrong keyboard layout:

- `יקךךם` → `hello`
- `akuo` → `שלום`

TypoSwitch maps keyboard keys between Hebrew and English and converts your text instantly.
